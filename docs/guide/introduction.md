# 介绍

## 什么是 Luna UI？

Luna UI 是一个基于 **Vue 3** + **TypeScript** 构建的现代化 UI 组件库，提供一套高质量、可定制的组件，帮助你快速搭建优美的 Web 应用。

## 特性

- 🎨 **主题定制** — 基于 CSS Variables 的主题系统，支持亮色 / 暗色 / 跟随系统模式
- 🧩 **Vue 3 + TypeScript** — 完整的类型支持，基于 Composition API 开发
- 📦 **按需引入** — 支持 Tree-shaking，只打包你使用的组件
- ⚡ **Vite 驱动** — 基于 Vite 构建，开发体验极速
- 🎯 **设计规范** — 统一的设计语言，一致的交互体验

## 组件总览

| 组件 | 说明 |
|------|------|
| [Button 按钮](/components/button) | 6 种颜色 × 4 种变体 × 3 种尺寸，支持 loading、圆角、全宽等 |

| 组件 | 说明 |
|------|------|
| [Input 输入框](/components/input) | 支持 v-model、可清除、前后缀插槽、校验状态 |
| [Switch 开关](/components/switch) | 支持 v-model、标签、loading 状态 |

## 设计理念

Luna UI 的设计遵循以下原则：

1. **简洁优先** — 组件 API 简洁直观，降低学习成本
2. **可定制** — 通过 CSS Variables 轻松定制主题，适配不同品牌风格
3. **一致性** — 统一的尺寸体系（sm / md / lg）、颜色体系和交互模式
4. **可访问性** — 关注键盘操作和语义化标签
